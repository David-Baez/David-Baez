# ***<a href="https://github.com/zhanghai/MaterialFiles"> Material Files </a>***

> Un administrador de archivos de diseño de Material Design de código abierto.
> 
> Características:
> - Fuente abierta: Ligero, limpio y seguro.
> - Material Design: sigue las pautas de Material Design, prestando atención a los detalles.
> - Breadcrumbs: navegue por el sistema de archivos con facilidad.
> - Soporte de root: vea y administre archivos con acceso de root.
> - Soporte de archivo: vea, extraiga y cree archivos comprimidos comunes.
> - Soporte NAS: vea y administre archivos en servidores SFTP y SMB.
> - Temas: colores de la interfaz de usuario personalizables, además del modo nocturno con negro verdadero opcional.
> - Compatible con Linux: conoce enlaces simbólicos, permisos de archivos y contexto SELinux.
> - Robusto: utiliza llamadas al sistema Linux bajo el capó, no otro analizador ls más.
> - Bien implementado: construido sobre las cosas correctas, incluyendo Java NIO2 File API y LiveData.
